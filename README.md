# ESP32-P4 电子鼻网关 (E-Nose Gateway)

基于 ESP32-P4 的高性能空气质量监测系统，通过 ESP-NOW + UART 桥接采集多节点传感器数据，实现本地可视化、云端同步与远程控制。

## 系统架构

```
┌─────────────────────────────────────────────────────┐
│                  云端服务器                          │
│    POST /api/sensor/data   GET /api/p4/poll          │
└──────────┬──────────────────────────┬────────────────┘
           │ HTTP                     │ HTTP
┌──────────▼──────────────────────────▼────────────────┐
│                   ESP32-P4 (主控)                     │
│  • LVGL 9.5 UI (1024×600)       • 数据上传/命令轮询   │
│  • 传感器数据融合与AI分析        • QR扫码配网          │
│  • WiFi自动重连/NVS凭证持久化    • 摄像头/音频          │
└─────────────────────┬────────────────────────────────┘
                      │ UART1 (GPIO32/36, 115200)
┌─────────────────────▼────────────────────────────────┐
│              ESP32-C6 (桥接器)                         │
│  • ESP-NOW 接收S3节点数据  • BLE备用通道              │
│  • WiFi STA (ESP-Hosted SDIO)  • 心跳检测             │
└──────────┬──────────────────────────┬────────────────┘
           │ ESP-NOW                  │ ESP-NOW
┌──────────▼──────────┐    ┌──────────▼──────────┐
│  S3-A 传感器节点      │    │  S3-B 传感器节点      │
│  VOC/CO2/HCHO/CO     │    │  VOC/CO2/HCHO/CO     │
│  温湿度/UV           │    │  温湿度               │
│  紫外线灯控制         │    │                      │
└─────────────────────┘    └─────────────────────┘
```

## 核心功能

### 传感器数据采集
- 多节点（S3-A/S3-B）传感器数据通过 ESP-NOW 实时上报至 C6 桥接器
- 支持 VOC、CO₂、HCHO、CO、温湿度、气味、UV 等指标
- C6 通过 UART1 透传数据至 P4，心跳检测保活（30s 超时）

### UI 交互 (LVGL 9.5)
- 主屏仪表盘：节点数据概览、状态栏、环境信息
- 控制中心：WiFi 管理、C6 桥接状态、云同步开关、亮度/音量调节
- 节点详情页：传感器实时数值、UV 灯控制、AI 分析结果
- QR 扫码：摄像头识别二维码快速配置设备
- 应用抽屉：页面间快速导航

### 云同步
- 传感器数据定时上传至云端（5s 间隔，可切换 1s 快速模式）
- 远程命令轮询：接收 set_uv / alert / reboot / report 等指令
- WiFi 断连自动重连，凭据 NVS 持久化

### AI 分析
- 集成云端大模型 API，对当前空气质量进行智能分析
- 识别空气等级并提供改善建议
- 支持远程触发 AI 分析

## 硬件配置

| 组件 | 型号/规格 |
|---|---|
| 主控 | ESP32-P4 (双核 RISC-V) |
| 桥接器 | ESP32-C6 (WiFi 6 + ESP-NOW) |
| 传感器节点 | ESP32-S3 + 多气体传感器模组 |
| 显示屏 | 1024×600 MIPI DSI, RGB565 |
| 触摸 | GT911 电容触摸 |
| 摄像头 | SC2336 (1280×720 MIPI) |
| 音频 | ES8311 编解码器 (I2S) |
| 环境传感器 | BME680 (温度/湿度/气压) |
| SD 卡 | microSD, SDMMC 接口 |
| 外部存储 | SPIFFS (存放音频资源) |

## 软件栈

- **ESP-IDF** 5.5.3
- **LVGL** 9.5.0
- **ESP-Hosted** (C6 通过 SDIO 提供 WiFi)
- **cJSON** (数据解析)
- **ESP-NOW** (节点通信)

## 快速开始

```bash
# 配置项目
idf.py set-target esp32p4
idf.py menuconfig

# 构建
idf.py build

# 烧录
idf.py flash monitor
```

### 关键配置
- 静态 IP: `192.168.110.100`
- 服务器地址: `http://*.*.*.*/api/` <!-- Censored / 已脱敏 -->
- UART1: GPIO32(TX) / GPIO36(RX), 115200 baud
