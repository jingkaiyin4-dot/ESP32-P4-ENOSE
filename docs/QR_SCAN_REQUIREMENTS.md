## 1. 概述

### 1.1 项目背景

本项目基于 ESP32-P4 Function EV Board 开发电子鼻系统主基站，负责：
- 通过摄像头扫描 ESP32-S3 负载屏幕上的二维码
- 识别不同 S3 设备的身份（ID）
- 在 7 寸屏幕上显示对应 S3 设备采集的传感器数据

### 3.3 科幻特效详情

#### 3.3.1 扫描框
- 四角矩形边框，颜色 #00D4FF
- 边框宽度：3px
- 角标长度：20px
- 带发光效果（glow shadow）

#### 3.3.2 扫描线
- 水平线，高度 2px
- 颜色：#00FFFF → #7B68EE 渐变
- 动画：从上到下循环移动，周期 2 秒
- 透明度：70%

#### 3.3.3 数据浮现面板
- 背景：半透明黑色 (rgba(0,0,0,0.7))
- 圆角：15px
- 边框：1px #00FFFF
- 内边距：15px
- 动画：从透明到可见，300ms ease-out

#### 3.3.4 边缘呼吸灯
- 屏幕边缘淡入淡出青色光晕
- 周期 3 秒

## 4. 技术实现规范

### 4.1 硬件配置

| 硬件 | 配置 |
|------|------|
| 主控 | ESP32-P4 |
| 摄像头 | MIPI-CSI 接口，SC2336 传感器 |
| 屏幕 | MIPI-DSI 7寸 1024x600 |
| 摄像头分辨率 | QVGA (320x240) - 省资源 |

### 4.2 软件架构

```
┌─────────────────────────────────────────────┐
│                  UI 层 (LVGL)                │
│  - scr_startup (开机界面)                    │
│  - scr_main (主界面)                        │
│  - scr_qr_scan (扫码界面)                   │
└─────────────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────┐
│              业务逻辑层 (App)                │
│  - WiFi 连接                                │
│  - 假数据生成 (已有)                        │
│  - 二维码识别控制                           │
└─────────────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────┐
│              驱动层 (BSP/ESP-IDF)           │
│  - 摄像头驱动 (esp_camera)                  │
│  - 显示驱动 (BSP Display)                   │
│  - WiFi 驱动 (esp_wifi)                     │
└─────────────────────────────────────────────┘
```

### 4.3 二维码识别组件

| 组件 | 说明 |
|------|------|
| quirc | 轻量级二维码解码库 |
| esp_video | 视频捕获 |
| esp_cam_sensor | 摄像头传感器驱动 |

### 4.4 任务设计

```c
// 二维码扫描任务
static void qr_scan_task(void *param) {
    while (1) {
        // 1. 获取摄像头帧
        // 2. quirc 解码
        // 3. 解析 JSON {"id": "S3_A"}
        // 4. 更新 UI
        // 5. 延时 100ms (10fps)
    }
}
```

### 4.5 二维码数据格式

```json
{"id": "S3_A"}
```

或

```json
{"id": "S3_B"}
```

### 4.6 数据匹配

| 二维码 ID | 显示设备 | 数据源 |
|-----------|---------|--------|
| S3_A | Node S3-A | 现有假数据 |
| S3_B | Node S3-B | 现有假数据 |
