# 🎬 智能嗅觉电子鼻系统解说视频拍摄与口播指南 (Video Production & Spoken Script Guide)

本指南专为 **"ESP32-P4 + S3 智能机器人嗅觉电子鼻网关系统"** 的展示与口播录制设计。视频总时长控制在 **3分钟 (180秒)** 左右，采用“痛点切入 -> 硬件展示 -> 核心演示 -> 底层黑科技 -> 总结”的黄金叙事结构。

---

## 📅 一、 拍摄前期准备与所需素材 (Production Checklist & Assets)

在开机拍摄前，请准备好以下**物理素材**与**软件素材**：

### 1. 🎬 物理实物与道具 (Physical Props)
*   **网关主控端**：运行正常 LVGL 9.5 UI 的 ESP32-P4 开发板（配 7 寸 MIPI DSI 屏幕、摄像头、ES8311 喇叭及外壳）。
*   **边缘节点端**：ESP32-S3 传感器终端（连接 ADS1115、气体传感器、SG90 开盖舵机、排风扇、紫色 UV 杀菌灯）。
*   **对比道具**：
    *   一颗**新鲜香蕉 / 芒果**（置信度高，绿色正常状态）。
    *   一颗**过度成熟/变质的香蕉 / 芒果**，或一瓶**酒精/挥发性溶剂**（用于快速让传感器数值飙升，触发告警）。
*   **手绘蓝图**：将您那张极具艺术感的**手绘电路机壳设计图纸**打印出来，或者在 iPad/电脑上全屏显示，作为转场或背景展示。

### 2. 💻 软件与界面素材 (Software Assets)
*   **网页管理后台**：在笔记本电脑上打开 Web Dashboard，展示设备状态、传感器历史曲线。
*   **模型训练文件**：在电脑上展示 TFLite 神经网络训练代码的 Jupyter Notebook 或模型结构图（如 Netron 可视化）。

## 🔄 二、 演示操作流程 (Demonstration Workflow)

建议录制现场按照以下四步法进行演示：
1.  **静态展示**：拉近镜头扫过精美的 3D 打印外壳、激光雕刻队名（**“麦当劳别涨价了队”**）与手绘图纸。
2.  **正常检测**：将新鲜水果放入检测仓，展示 P4 屏幕上显示“新鲜度 95% + 状态：绿色良好”。
3.  **变质突变 (高潮)**：迅速移入变质水果或用棉签蘸取少许酒精靠近传感器，数值瞬间暴涨，屏幕变红告警。
4.  **物理闭环联动**：展示 UV 消毒灯瞬间点亮，舵机驱动密封盖“咔哒”一声弹开，风扇开始极速旋转排气。
5.  **多模态交互**：按下屏幕触发 AI 诊断，喇叭中传出清晰的流式语音建议。

## 🎙️ 四、 专业口播录制与调试建议 (Professional Recording Tips)

为了保证视频达到专业级的展示效果，请参考以下调试建议：

> [!IMPORTANT]
> **屏幕防闪烁与抗摩尔纹 (Camera Setting Guide)**
> *   **调整相机快门速度**：在用手机或相机拍摄 7 寸 DSI 屏幕时，如果快门速度不对，屏幕会出现滚动的黑色横条（频闪）。请将相机快门速度固定为 **1/50 秒** 或 **1/100 秒**（对应国内 50Hz 光源和液晶刷新率），以获得完美清晰的屏幕画面。
> *   **避免强光直射**：拍摄时请微调光线角度，避免反光板或顶灯在屏幕上形成白斑，影响 LVGL 仪表盘界面的文字清晰度。

> [!TIP]
> **音频同期录制与降噪 (Audio Guide)**
> *   **同期声清脆质感**：在演示第四步（舵机开盖、风扇转动、继电器吸合）时，建议保留现场的机械声（如舵机的“咔哒”声），这在技术演示视频中非常具有真实感和说服力。
> *   **口播人声加麦**：录制解说词时，尽量使用领夹麦克风或在安静房间内后期配音，避免开发板风扇的底噪掩盖了解说人声。
